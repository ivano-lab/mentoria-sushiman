Aula 2

Passos do Projeto

Obs.: Partindo-se do pressuposto que o repositório já foi previamente forkado e o git clone efetuado, vamos aos seguintes passos...

Passo 1: Configuração do Ambiente

Antes de começar, você precisará ter o Python e o Django instalados em seu sistema. Você pode instalar o Django usando o pip:

pip install django

Passo 2: Criar um Projeto Django

Crie um novo projeto Django usando o seguinte comando:

django-admin startproject cardapio-virtual

Passo 3: Configurar o Django para o Front-End

No seu aplicativo, crie uma pasta chamada templates para armazenar os templates HTML.

Passo 4: Criar Páginas HTML e Usar Materialize CSS (CDN)

Dentro da pasta templates do seu aplicativo, crie arquivos HTML para suas páginas, como index.html, sobre.html, etc.

Passo 5: Configurar URLs e Views

Configure as URLs do seu aplicativo no arquivo urls.py do seu aplicativo.

Passo 6: Executar o Servidor de Desenvolvimento

No diretório raiz do seu projeto Django, execute o servidor de desenvolvimento:

python manage.py runserver

Acesse o aplicativo em seu navegador em http://localhost:8000 e você verá suas páginas HTML usando o Materialize CSS.

A partir daqui, você pode adicionar mais funcionalidades, modelos de dados, formulários e interações JavaScript conforme necessário para o seu projeto. O Django é altamente personalizável e pode ser estendido para atender às necessidades do seu aplicativo web.
